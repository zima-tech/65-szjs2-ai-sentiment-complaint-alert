# AI舆情分析与投诉预警

## 原需求描述

实时抓取清江路或文二西路周边社区论坛、社交媒体及政府投诉平台关于项目的信息，利用自然语言处理技术进行情感分析和舆情分类。当出现"夜间施工"、"扬尘投诉"等负面舆情苗头时，系统自动预警并建议采取相应措施（如调整作业时间、加强洒水降尘）。

---

## 页面结构

```
- 舆情分析首页
  - 舆情概览
  - 预警提示

- 舆情监控页
  - 数据来源
  - 实时数据

- 情感分析页
  - 情感分布
  - 趋势变化

- 预警管理页
  - 预警规则
  - 预警记录
```

## 功能列表

### 1. 数据采集

- 监控来源：
  ```
  ├─ 社区论坛
  ├─ 社交媒体
  ├─ 政府投诉平台
  └─ 新闻媒体
  ```

- 关键词配置
- 实时爬取

### 2. 舆情分类

- 自动分类：
  ```
  [舆情分类]

  ├─ 正面舆情
  │  └─ 表扬、感谢
  │
  ├─ 中性舆情
  │  └─ 询问、建议
  │
  └─ 负面舆情
     ├─ 夜间施工投诉
     ├─ 扬尘投诉
     ├─ 噪音投诉
     └─ 交通影响投诉
  ```

### 3. 情感分析

- 情感倾向判断：
  ```
  [情感分析]

  来源：XX社区论坛
  内容：晚上施工太吵了，睡不着觉

  情感分析：
  ├─ 情感倾向：负面
  ├─ 情感强度：-0.85
  ├─ 关键词：夜间施工、噪音
  └─ 涉及项目：清江路工程

  [需重点关注]
  ```

### 4. 自动预警

- 预警规则配置
- 预警触发：
  ```
  [舆情预警]

  预警等级：橙色
  预警类型：夜间施工投诉
  来源：XX投诉平台
  时间：2024-01-15 22:30

  内容摘要：
  市民反映清江路工地夜间施工，
  噪音严重影响居民休息。

  建议措施：
  ├─ 核实夜间施工审批情况
  ├─ 调整作业时间安排
  ├─ 加强降噪措施
  └─ 安排人员与居民沟通

  [立即处理] [查看详情]
  ```

### 5. 措施建议

- 自动生成应对建议
- 责任部门推送

## 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| source | string | 来源 |
| content | string | 内容 |
| sentiment | enum | 情感倾向 |
| sentiment_score | float | 情感得分 |
| category | enum | 舆情分类 |
| alert_level | enum | 预警等级 |
| suggestion | string | 建议措施 |
